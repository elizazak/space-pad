# Space Pad

Interactive space sound pad with falling stars animation. Works on desktop and mobile.

## Features
- 16 colorful pads that light up when played
- Falling stars background animation
- Record up to 3 sound configurations (play/delete)
- Responsive design

## Usage
- Click a pad to play sound
- Use recording controls to play or delete
- Record button hidden on very small screens

## Responsiveness
- ≤1024px: tablets & small laptops
- ≤480px: smaller fonts & elements
- ≤380px: pads aligned to top (`flex-start`)
- ≤320px: record button hidden
